核心能力

不只是聊天，能自主规划、调用工具、完成复杂任务

自进化

跑完复杂流程后，阿布主动提议「固化成技能」，一键生成草稿，审阅上架后下次直接调用。越用越专属于你的工作流。

Agent 自主执行

自主规划任务、调用工具、读写文件、执行命令，完成复杂工作流。

Skill 技能系统

内置 28 个技能，覆盖 Office 文档、设计创作、浏览器自动化、开发辅助等场景，一键安装，支持自定义扩展。

MCP 工具协议

通过 Model Context Protocol 连接数据库、搜索引擎、GitHub 等外部服务。

IM 频道集成

接入飞书、钉钉、企业微信、Slack 等 IM 平台，@阿布即可团队协作。

触发器 / 值班

HTTP、文件变更、Cron、IM 消息等事件自动触发，实现 7×24 无人值守。

电脑操控

截屏识别 + 键鼠控制，让阿布直接操作桌面应用，像真人一样完成任务。

沙箱安全

macOS Seatbelt 沙箱隔离 + 敏感路径保护 + 命令安全检查，本地数据不上传。

多模型支持

支持 Claude、DeepSeek、通义千问、豆包、Moonshot、智谱等主流模型，以及 Ollama 本地模型零配置接入。

Projects 管理

工作区可升级为 Project，相关对话自动聚合归类，每个项目独立配置默认模型、技能集、MCP，一次设置长期生效。

对话分享

把与阿布的对话打包成 .abuShare 文件发给朋友，对方导入后即可查看完整记录，AI 生成的附件也一并打包带走。

智能通知

任务完成、消息提醒通过菜单栏图标和侧边栏角标推送；全屏或勿扰时暂存 inbox，回到窗口可见未读提醒。

性格系统（Soul）

三档主动度预设（寡言 / 伙伴 / 管家）控制阿布何时主动开口；用 SOUL.md 自定义语气、称呼、回复风格与边界。

内容安全扫描

自动扫描 agent 写入的 skill / 记忆，拦截危险指令、prompt 注入等 120+ 类风险；三档权限模式按场景调节松紧。

诊断面板

设置 → 诊断，逐项自检 AI 服务、MCP、技能、网络、应用环境；出问题一键导出诊断包，API Key 与路径自动脱敏。
